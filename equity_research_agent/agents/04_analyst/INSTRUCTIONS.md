# Agent 04: Analyst (분석 에이전트)

## 역할

검증된 데이터를 기반으로 파생 인사이트를 도출합니다. 단순 수치 나열을 넘어 AI Capex, 공급망 재편, 정책적 변화와 이익 간의 상관관계를 분석합니다.

## 입력

- `validated_data/`: 03_data_validator의 정제 데이터
- 매크로 경제 컨텍스트 정보

## 출력

- `analysis_results/`: 분석 결과 및 인사이트
- `insights.json`: 구조화된 인사이트 데이터

## 분석 프레임워크

### Framework 1: 반도체/AI 사이클 분석

미국의 대규모 AI Capex 투자가 아시아 반도체 기업의 이익으로 전이되는 인과관계를 분석합니다.

#### 인과관계 체인

```
┌─────────────────────────────────────────────────────────────────┐
│                    AI Capex → Earnings 전이 구조                 │
└─────────────────────────────────────────────────────────────────┘

[미국 빅테크 AI Capex]
     │
     ▼
┌─────────────┐
│ GPU 수요 ↑  │ → NVIDIA 매출 ↑
└──────┬──────┘
       │
       ▼
┌──────────────────────────────────────────────────────────────┐
│                     HBM 수요 폭발적 증가                       │
├──────────────┬──────────────┬───────────────────────────────┤
│  SK하이닉스   │   삼성전자    │         Micron             │
│  HBM3E 선도  │  HBM 추격    │      미국 진출             │
│  EBITDA +65% │             │                              │
└──────────────┴──────────────┴───────────────────────────────┘
       │
       ▼
┌──────────────────────────────────────────────────────────────┐
│                    파운드리 가동률 상승                        │
├───────────────────────────┬──────────────────────────────────┤
│          TSMC             │         삼성 파운드리            │
│    선단 공정 Full Capa    │        점유율 회복 시도          │
│    EBITDA +22%           │                                  │
└───────────────────────────┴──────────────────────────────────┘
```

#### 분석 지표

| 지표 | 소스 | 의미 |
|------|------|------|
| US Big Tech Capex | 분기 실적 | AI 투자 규모 선행 지표 |
| HBM 출하량 | TrendForce | 메모리 수요 지표 |
| 파운드리 가동률 | 업계 리포트 | 생산 Capa 활용도 |
| ASP 추이 | 기업 실적 | 가격 결정력 |

#### 이익 전이 모델

$$
\Delta EBITDA_{Korea} = \alpha \cdot \Delta Capex_{US} + \beta \cdot \Delta HBM_{ASP} + \gamma \cdot \Delta DRAM_{Utilization}
$$

**파라미터 추정**:
- α: AI Capex 민감도 (0.3~0.5)
- β: HBM 가격 민감도 (0.4~0.6)
- γ: 가동률 민감도 (0.2~0.3)

### Framework 2: 중국 정책 요인 분석 (Anti-Involution)

중국 정부의 '안티-인볼루션(Anti-Involution)' 정책이 기업 마진과 ROE 회복에 미치는 영향을 분석합니다.

#### 인볼루션(内卷) 문제

```
┌─────────────────────────────────────────────────────────────┐
│                    인볼루션 악순환 구조                       │
└─────────────────────────────────────────────────────────────┘

[과잉 경쟁] → [가격 전쟁] → [마진 붕괴] → [R&D 투자 위축]
     ↑                                           │
     └───────────────────────────────────────────┘
```

#### 안티-인볼루션 정책 효과

| 정책 | 대상 섹터 | 예상 효과 |
|------|-----------|-----------|
| 가격 덤핑 규제 | 전기차, 태양광 | 마진 회복 |
| 보조금 합리화 | 신에너지 | 비효율 기업 퇴출 |
| 공급 과잉 억제 | 철강, 화학 | 가격 안정화 |

#### 분석 지표

```yaml
china_policy_indicators:
  - name: "Manufacturing Profit Margin"
    current: 4.5%
    historical_avg: 6.2%
    recovery_target: 5.5%

  - name: "Industry Concentration (CR4)"
    trend: "increasing"
    implication: "pricing_power_recovery"

  - name: "Loss-making SOE Ratio"
    current: 25%
    target: 15%
```

#### ROE 회복 시나리오

$$
ROE = \frac{Net\,Income}{Equity} = Margin \times Turnover \times Leverage
$$

**시나리오 분석**:
| 시나리오 | 마진 변화 | ROE 영향 |
|----------|-----------|----------|
| Base | +1%p | ROE +2%p |
| Bull | +2%p | ROE +4%p |
| Bear | +0.5%p | ROE +1%p |

### Framework 3: 인도 내수 모멘텀 분석

인도의 GST 세제 개편 및 정부 자본 지출 확대가 내수 소비재 및 금융 이익에 기여하는 바를 분석합니다.

#### 인도 성장 드라이버

```
┌─────────────────────────────────────────────────────────────┐
│                    인도 성장 삼각 구조                        │
└─────────────────────────────────────────────────────────────┘

           ┌──────────────────┐
           │   정부 Capex ↑   │
           │  (인프라 투자)   │
           └────────┬─────────┘
                    │
        ┌───────────┴───────────┐
        ▼                       ▼
┌───────────────┐       ┌───────────────┐
│  고용 창출    │       │  건설/소재    │
│  소득 증가    │       │  수요 증가    │
└───────┬───────┘       └───────┬───────┘
        │                       │
        └───────────┬───────────┘
                    ▼
           ┌───────────────┐
           │   소비 확대   │
           │  (내수 성장)  │
           └───────────────┘
```

#### 핵심 분석 지표

| 지표 | 2024 | 2025E | 2026E | 의미 |
|------|------|-------|-------|------|
| GDP 성장률 | 7.0% | 6.8% | 7.2% | 전체 성장 기조 |
| 정부 Capex/GDP | 3.2% | 3.5% | 3.8% | 재정 투입 강도 |
| 신용 성장률 | 16% | 15% | 14% | 금융 섹터 모멘텀 |
| 소비재 매출 성장 | 12% | 14% | 15% | 내수 수요 |

#### 섹터별 이익 기여도

```yaml
india_sector_contribution:
  Financials:
    weight: 35%
    eps_growth: 18%
    drivers:
      - "신용 성장 지속"
      - "NIM 안정화"
      - "자산 건전성 개선"

  Consumer Staples:
    weight: 12%
    eps_growth: 15%
    drivers:
      - "농촌 소비 회복"
      - "GST 간소화"
      - "유통 확대"

  Industrials:
    weight: 10%
    eps_growth: 20%
    drivers:
      - "정부 인프라 발주"
      - "PLI 스킴 혜택"
      - "수출 경쟁력"
```
